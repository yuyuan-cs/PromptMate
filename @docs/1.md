## 💡 开发建议

### 保持ID唯一性
```typescript
// 使用时间戳或UUID确保ID唯一
id: `custom-${Date.now()}`
// 或
id: crypto.randomUUID()
```

### 使用变量占位符
```typescript
content: "请帮我分析{{主题}}，重点关注{{重点}}，输出格式为{{格式}}"
```

### 分类一致性
确保提示词的 `category` 字段与 `defaultCategories` 中的 `id` 匹配。
